# Fourier Series Visualization (Python + Pygame)

Интерактивная визуализация рядов Фурье с использованием эпицирклов.  
Проект демонстрирует, как сумма гармоник формирует сложные периодические функции — прямоугольную и пилообразную.  
Анимация выполнена на **Python** с использованием **Pygame**.

# Описание проекта

Проект визуализирует построение ряда Фурье через вращающиеся окружности (эпицирклы).  
Каждая гармоника представляется окружностью, на которой вращается точка.  
Последовательное суммирование гармоник формирует:

- прямоугольную волну;
- пилообразную волну.

В правой части экрана отображается формирующаяся во времени волна — как на осциллографе.

Приложение интерактивно — пользователь может менять параметры ряда прямо во время работы программы.

# Особенности

### ✔ Ряды Фурье:
- Square wave (прямоугольная)
- Sawtooth wave (пилообразная)

### ✔ Эпициркловая анимация:
- каждая гармоника — отдельный вращающийся круг
- реалистичная визуализация вклада каждой гармоники

### ✔ Полный контроль пользователя:
- изменение числа членов ряда
- регулировка скорости вращения
- смена типа функции
- пауза, сброс, выход

### ✔ Гибкая конфигурация:
Все параметры хранятся в `config.json`:
- размеры окна
- позиция эпицирклов
- скорость волны
- масштаб окружностей
- цвета интерфейса

### ✔ Красивый визуальный стиль:
- глубокий тёмный фон  
- мягкие голубые окружности  
- яркая красная волна  
- аккуратные вспомогательные линии

# Математическая основа

## Прямоугольная функция (Square Wave)

Ряд Фурье для прямоугольного сигнала:

$$
f(t) = \frac{4}{\pi} 
\sum_{k=0}^{\infty} \frac{1}{2k+1} \sin((2k+1)t)
$$

Используются только нечётные гармоники:  
1, 3, 5, 7, …

---

## Пилообразная функция (Sawtooth Wave)

Ряд Фурье:

$$
f(t) =
\frac{2}{\pi}
\sum_{n=1}^{\infty}
\frac{(-1)^{n+1}}{n} \sin(nt)
$$

В отличие от прямоугольной волны используются **все гармоники**.

---

## Эпицирклы

Каждая гармоника визуализируется вращающейся окружностью:

$$
x_{n+1} = x_n + A_n \cos(\omega_n t)
\]
\[
y_{n+1} = y_n + A_n \sin(\omega_n t)
$$

где:

- $$\(A_n\)$$ — радиус вклада гармоники,  
- $$\(\omega_n\)$$ — частота гармоники,  
- $$\(t\)$$ — время,  
- $$\(x_n, y_n\)$$ — координаты центра текущей окружности.
